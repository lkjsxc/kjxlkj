# Compact Rail Layout

## Narrow Layout Rule

- Narrow note and history rails keep the same one-column timeline order as wide rails.
- `Prev` stays above `Next`.
- Compact drawers do not collapse the timeline into a side-by-side row.

## Stability Rules

- Timeline cards keep stable footprints even when disabled.
- Labels remain outside the cards on narrow screens.
- Compact rail navigation must stay readable without page-level horizontal overflow.
- Main-pane pager layout is controlled separately by [../paging/compact-pager-layout.md](../paging/compact-pager-layout.md).

# Stacked Shell Contract

## Narrow Layout

- Narrow screens keep the rail visible by stacking it above the main pane.
- The rail remains part of normal document flow instead of becoming an overlay.
- The page must remain fully usable without opening or closing navigation chrome.

## Rail Rules

- Rail sections keep the same order as desktop.
- Search stays inside the rail.
- Recent-note shortcuts remain visible above the fold when practical.

## Anti-Regression Rules

- No drawer backdrop.
- No floating menu button.
- No floating close button.

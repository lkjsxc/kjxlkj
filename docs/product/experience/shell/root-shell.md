# Root Shell Contract

## Global Rule

- Every HTML page renders inside the same shell structure.
- The side menu is always visible on root, admin, note, and history pages.
- The shell never collapses into a drawer in this phase.

## Rail Content

- Brand and session mode.
- Search form for the current scope.
- Scope summary such as `Public index`, `Admin index`, or current note context.
- Recent accessible notes ordered by recency.
- Mode-specific text actions.

## Main Pane

- Root and admin pages keep the dense scalable list in the main pane.
- Note and history pages keep document-focused content in the main pane.
- The rail adds navigation and context without taking over result browsing.

# Docs Layout Contract

## Top-Level Docs Directories

- `docs/vision/`
- `docs/product/`
- `docs/architecture/`
- `docs/containers/`
- `docs/operations/`
- `docs/repository/`
- `docs/restructuring/`

## Recursive Rule

- Every docs subdirectory must include exactly one `README.md` TOC.
- TOCs must link to all child docs and child TOCs.

# Conformance Sub-Ledgers

Back: [/docs/reference/README.md](/docs/reference/README.md)

Domain-specific conformance ledgers used to support `/docs/reference/CONFORMANCE.md`.

## Rule

Each sub-ledger records evidence-backed current status for its domain and must not contain target-only claims without explicit `unverified` marking.

# Scope Boundaries

## In Scope

- Full rewrite of `docs/restructuring/**` with recursive TOC structure.
- Stage and wave decomposition from `S00` through `S10`.
- Checkpoint, evidence, and migration contracts for restructuring.
- Cross-link updates in relevant `docs/**` files that reference restructuring.

## Out of Scope

- Broad rewrites of non-restructuring documentation domains.
- Runtime behavior changes unrelated to documentation contracts.
- Any backward-compatibility aliasing of old restructuring paths.

## Guardrails

- One `README.md` per directory.
- Multiple children under every docs directory.
- Stable heading schemas for machine parsing.
- Relative links only.

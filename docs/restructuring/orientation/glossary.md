# Restructuring Glossary

## Terms

- **Stage**: Major restructuring slice with 3 waves and stage-level exit criteria.
- **Wave**: Smallest ordered execution unit with explicit acceptance IDs.
- **Gate**: Mandatory verification command or audit step.
- **Evidence**: Structured output proving gate or stage outcome.
- **Blocked**: Non-pass state caused by unmet prerequisites.
- **Drift**: Mismatch between intended and current contract surfaces.
- **Cutover**: Point where old restructuring paths are replaced by new paths.

## Identifier Conventions

- Stage IDs: `S00` .. `S10`
- Wave IDs: `W000` .. `W102`
- Stage exit IDs: `Sxx-EXIT-yy`
- Wave acceptance IDs: `Sxx-Wyyy-ACzz`

# Redirect and Link Policy

## Policy

- Do not preserve legacy restructuring links as compatibility aliases.
- Update all in-repo references to new canonical paths in the same change set.
- Reject ambiguous references to removed phase files.

## Allowed Reference Forms

- Relative links to canonical files.
- TOC entries in parent `README.md` files.

## Disallowed Forms

- External URLs as primary references for current restructuring contracts.
- Links to deleted legacy restructuring files.
- Inline references without a canonical path.

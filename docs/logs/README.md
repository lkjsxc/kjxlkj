# Execution Logs

Back: [/docs/README.md](/docs/README.md)

Dated execution logs, audits, and improvement journals.

## Structure

Logs are organized by date: `YYYY/MM/DD/`.

Each date directory contains:

- `README.md` — summary of what was done
- `audit.md` — decisions, deviations, and evidence
- `improvement-ideas.md` — ideas for future improvement
- `todo-progress.md` — TODO checkpoint status

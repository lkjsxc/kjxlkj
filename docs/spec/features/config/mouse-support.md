## Click Actions

### Single Click

| Location | Action |
|----------|--------|
| Text area | Move cursor |
| Line number | Select line |
| Fold column | Toggle fold |
| Sign column | Show diagnostics |
| Tab | Switch tab |

### Double Click

| Location | Action |
|----------|--------|
| Word | Select word |
| Number | Select number |
| String | Select string |

### Triple Click

| Location | Action |
|----------|--------|
| Line | Select entire line |

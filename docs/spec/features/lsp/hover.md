# Hover Documentation

View documentation by hovering over symbols.

## Usage

### Keybinding

| Key | Action |
|-----|--------|
| `K` | Show hover |
| `gh` | Show hover (alternative) |

### Hold Position

Hover popup stays while key held or until dismissed.

## Navigation

### Within Hover

| Key | Action |
|-----|--------|
| `j` | Scroll down |
| `k` | Scroll up |
| `<C-d>` | Page down |
| `<C-u>` | Page up |
| `<Esc>` | Close |

## Multiple Hovers

### On Same Position

Some locations have multiple hover results.

| Key | Action |
|-----|--------|
| `]h` | Next hover |
| `[h` | Previous hover |

## LSP Integration

### Requirements

Requires LSP server with hover support.

### Fallback

Without LSP, shows type from syntax tree.

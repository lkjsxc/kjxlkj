# Buffer Features

Buffer management and navigation features.

## Common Operations

| Key/Command | Action |
|-------------|--------|
| `:bn` | Next buffer |
| `:bp` | Previous buffer |
| `:b#` | Alternate buffer |
| `:bd` | Delete buffer |
| `:ls` | List buffers |
| `Ctrl-^` | Toggle alternate |

## Buffer States

| State | Meaning |
|-------|---------|
| Active | Displayed in window |
| Hidden | Loaded but not visible |
| Modified | Has unsaved changes |
| Readonly | Cannot be modified |

## How It Works

### Automatic Assignment

When you switch buffers, the previous
buffer becomes the alternate file.

### Flow Example

1. Open `main.rs` (current)
2. Open `lib.rs` (current: lib.rs, alt: main.rs)
3. `<C-^>` → back to `main.rs`
4. `<C-^>` → back to `lib.rs`

## Commands

| Command | Description |
|---------|-------------|
| `:BufferNext` | Next buffer |
| `:BufferPrev` | Previous buffer |
| `:BufferClose` | Close current |
| `:BufferCloseLeft` | Close all left |
| `:BufferCloseRight` | Close all right |
| `:BufferOnly` | Close all others |
| `:BufferPick` | Pick with letters |

## Navigation

### Between Windows

| Key | Direction |
|-----|-----------|
| `Ctrl-W h` | Left |
| `Ctrl-W j` | Down |
| `Ctrl-W k` | Up |
| `Ctrl-W l` | Right |

### Cycling

| Key | Action |
|-----|--------|
| `Ctrl-W w` | Next window |
| `Ctrl-W W` | Previous window |
| `Ctrl-W p` | Previous (last) |

## Resizing

### Increase/Decrease Height

| Key | Action |

| `Ctrl-W -` | Decrease height |

### Increase/Decrease Width

| Key | Action |
|-----|--------|

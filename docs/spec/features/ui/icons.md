# Icons and Glyphs

Visual symbols and icons.

## Overview

Icons enhance visual recognition of file
types, git status, and UI elements.

## Requirements

### Nerd Fonts

Icons require a Nerd Font patched terminal font:

- FiraCode Nerd Font
- JetBrains Mono Nerd Font
- Hack Nerd Font

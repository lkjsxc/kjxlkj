# Notification System

User notifications and alerts in kjxlkj.

## Notification Types

| Type | Duration | Interruption |
|------|----------|--------------|
| Info | 3s | None |
| Warning | 5s | None |
| Error | Persistent | Click to dismiss |
| Action | Persistent | Requires response |

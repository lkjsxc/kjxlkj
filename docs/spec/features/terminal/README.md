| Terminal navigation | Vi-like scrollback navigation |
| Exit handling | Clean process termination |

## Keybindings

| Key | Action |
|-----|--------|
| `<leader>t` | Toggle terminal |
| `Ctrl-\\ Ctrl-n` | Terminal to Normal mode |

Syntax highlighting and semantic features.

## Highlighting Sources

| Source | Description |
|--------|-------------|
| Tree-sitter | Fast incremental parsing |
| Semantic tokens | LSP-provided token types |
| Regex patterns | Fallback highlighting |

## Folding Keys

| Key | Action |
|-----|--------|
| `za` | Toggle fold |
| `zc` | Close fold |
| `zo` | Open fold |
| `zM` | Close all folds |
| `zR` | Open all folds |

# Navigation Features

Built-in navigation and search features.

## Quick Access

| Key | Feature |
|-----|---------|
| `<leader>f` | Fuzzy file finder |
| `<leader>g` | Live grep search |
| `<leader>e` | File explorer |
| `Ctrl-o` | Jump back |
| `Ctrl-i` | Jump forward |
| `g;` | Previous change |
| `g,` | Next change |

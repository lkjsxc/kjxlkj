## Overview

## Include Search

### Keybindings

| Key | Action |
|-----|--------|
| `[i` | Display first include line |
| `]i` | Display next include line |
| `[I` | List all includes |
| `]I` | List includes from cursor |

## Define Search

### Keybindings

| Key | Action |
